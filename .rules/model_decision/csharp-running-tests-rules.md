
## Usage guideline
When testing C# codes, use `csharp-language-plugin:test-csharp-code` skill, rather than `dotnet test` or other command line tools.

## Skill Description
`csharp-language-plugin:test-csharp-code` skill runs c# tests, generating test reports as well as coverage reports.

## Filter Flag
Always use `--filter` flag for fast test execution.
